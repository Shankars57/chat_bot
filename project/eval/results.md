# Offline Chatbot Evaluation Results

This file is generated by `chatbot.py`.

Run the script from the `project/` directory to create fresh zero-shot and one-shot results for all queries:

```bash
python chatbot.py
```

After execution, this file will contain:

- 20 e-commerce customer queries
- Zero-shot response for each query
- One-shot response for each query
- Heuristic scores for relevance, coherence, and helpfulness

Expected section format:

```text
## Query 1
Customer Query: How do I track my order?

Zero-shot Response:
...

One-shot Response:
...

---------------------------------
```
